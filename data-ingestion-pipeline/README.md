# Data ingestion pipeline

Includes Python modules for Data ingestion from WCA DB to another DB,
this can be used for filtering data on an specific country.

# Important

> This information is based on competition results owned and maintained by the
> World Cube Assocation, published at https://worldcubeassociation.org/results
> as of July 21, 2025.

This data pipelines works on v1.0.0 of WCA DB.
